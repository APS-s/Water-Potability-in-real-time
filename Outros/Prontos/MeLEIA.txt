# Explicação da nomeclatura dos modelos
modelo_agua => Significa que o modelo foi treinado com a base de dados de água
0.wxyz => Indica a % de precisão do modelo na medição da potabilidade da água
RMSprop || Adam || Adagrad || etc.. => Otimizador utilizado para treinar o modelo
x_Colunas => Quantidade de colunas utilizadas para treinar o modelo
    Se for 3:
        0 => pH
        1 => Condutividade Eletrica
        2 => Turbidez
    Se for 4:
        0 => pH
        1 => Condutividade Eletrica
        2 => Turbidez
        3 => Sólidos Totais
